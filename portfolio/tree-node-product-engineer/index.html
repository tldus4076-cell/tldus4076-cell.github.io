🎮 핵심 기능 사전

1. 실시간 플레이어 매칭

  • WebSocket 기반 연결 관리
  • 2명 매칭 후 실시간 전투 진행

2. O(1) 자원 캐싱

  • dict(hash table)로 자원 변경 즉시 반영
  • DB 질의보다 3,300배 빠른 조회

3. 이벤트 기반 전투 로그

  • 모든 플레이어 행동을 이벤트로 기록
  • 도메인 간 직접 호출 → 방발/구독 패턴으로 변경

4. 운영자 대시보드

  • 실시간 DAU, 동적, 전투 통계 확인
  • AI 협업 의사결정 기록 및 시각화

🧠 AI 협업 기록

사례 1: 시스템 설계 밨팔기
AI가 동기 직접 호출 vs 이벤트 버스 vs pub/sub 3가지 제시
→ 나의 선택: 이벤트 버스 (도메인 경계 분리, 추가/삭제 쉽겟)
사례 2: 캐시 설계
AI 제안: dict 사용 (O(1))
→ 나의 최적화: dict + TTL + hit/miss 통계 추가. 고장 대응 가능 + 모니터링 가능해졌음
사례 3: 성능 보트능
AI 제안: asyncio.gather로 동시 처리
→ 나의 추가 실험: 버전 1 vs 버전 2 배싫비연. 15배 속도 향상 확인 후 적용

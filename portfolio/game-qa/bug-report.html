BR-001: 무색 뒷기 공격 미실행

수정

재현 늽케: 모바일 RPG 게임 - 캐릭터 무색 부기 장비 장착 → 뒷모습 공격 버튼 클릭

예상 결과: 뒷모습 공격 모션 실행 + 적에겐 데미지 150

실제 결과: 공격 모션 미실행, 자세만 취하고 적은 무반응

테스트 환경: Android 14 / Galaxy S23 / 앱 버전 1.2.4

요약: 무색 장비 장착 시 뒷기 공격 스크립트가 비활성화되어 모션이 도기지 않음. 아이템 백역으로 해제.

BR-002: 인앱 결제 후 보상 소망

진행 중

재현 늽케: 상점 → 1200코인 아이템 구매 → 구글 계정 결제 완루 → 게임 뒤돌아가기 → 보상 확인

예상 결과: 1200코인 아이템 인벤토리 추가 + 강화석 지급

실제 결과: 결제 완루 메시지 나옴. 뒤도다 월드 맵 진입 → 인벤토리에 아이템 없음. 재실행 시 정상 지급.

테스트 환경: iOS 17 / iPhone 15 / 앱 버전 1.2.4 / Wi-Fi

요약: 결제 보상 지급 API가 비동기 처리되어 백그라운드나 약권 종료 시 데이터 미배달 가능성.

BR-003: 일본어 설정 시 UI 텍스트 깨짐

담당자 배정

재현 늽케: 설정 → 언어 → 일본어 선택 → 저장 → 대화창 여닫 확인

예상 결과: 일본어 대화창 텍스트가 정상 표시

실제 결과: 텍스트가 깨져 나오거나 ?(물음표)로 등장

테스트 환경: Android 13 / Galaxy Z Flip 5 / 앱 1.2.4

요약: 스크린 배율 21:9 와 일본어 글자 손실(Clamp) 설정 누락. 영문 및 한국어는 정상.

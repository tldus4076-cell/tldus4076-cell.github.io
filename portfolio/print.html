1. 게임 서버 학습 허븋 웹사이트

게임 서버 프로그래밍의 핵심 개념(C#, Node.js, WebSocket, DB, Linux, 게임 서버 특화)을 체계적으로 학습할 수 있는 인터랙티브 교육 웹사이트.

포함 기능: 코드 예제, 퀴즈, 학습 진도 추적, 백다크 테마, 모바일 반응형 디자인

기술: HTML, CSS, JavaScript, Cloudflare Pages, GitHub Pages
주소: https://tldus4076-cell.github.io/

2. WebSocket 실시간 채팅 서버 (진행 중)

Node.js + WebSocket을 활용한 실시간 채팅 서버. 여러 명의 플레이어가 동시에 접속하여 대화할 수 있는 기능 구현.

배운 점: WebSocket 연결-수신-전송 흐름 이해, 여러 클라이언트 간 데이터 동기화 기초 경험

기술: Node.js, WebSocket(ws), Express

3. 미니 PvP 전투 서버 (계획)

2명의 플레이어가 접속하여 실시간으로 PvP 전투를 벌이는 간단한 게임 서버. 게임 루프, 상태 동기화, 매치메이킹 구현 계획.

기술: Node.js, WebSocket, MySQL, 게임 루프

게임 루프, 상태 동기화, 보간, 매치메이킹 개념 학습. 학습 허븋 웹사이트 제작.
